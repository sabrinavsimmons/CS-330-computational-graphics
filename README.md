# CS-330-computational-graphics

This repository contains my final project for CS-330 Computational Graphics and Visualization at SNHU. The project is a 3D desktop workspace scene built in OpenGL and C++, featuring a monitor, keyboard, mouse, coffee mug, and pencil holder with textures, lighting, and camera navigation.

How do I approach designing software?

Working on this project helped me learn how to break complex real-world objects into simple geometric shapes. My design process began with examining the reference photo and figuring out which basic shapes could represent each object. I built each object step by step, testing one piece at a time before moving on to the next. This iterative process of planning, building, testing, and refining is something I will apply to future projects, no matter the field.

How do I approach developing programs?

I used a modular development strategy for this project. Each object had its own render function. This made debugging simpler because I could focus on one object at a time. Iteration played a key role in my process, with every milestone building on the previous one. Feedback from each submission influenced the next version. My approach shifted from writing everything in one place to thinking in terms of reusable, organized functions.

How can computer science help me in reaching my goals?

Computational graphics gave me knowledge and skills I can apply in both my educational and professional pathways. Educationally, understanding how 3D scenes are built using mathematics, transformations, and rendering pipelines gives me a stronger foundation for future courses that involve systems programming, simulation, or data visualization. Professionally, knowing how visual data is constructed and displayed opens doors in fields that rely on dashboards, data representation, and user interfaces. This course also reinforced skills like writing modular code, debugging systematically, and working through complex problems iteratively, all of which are directly applicable no matter what technical role I pursue.
